# Homepage section images

Add images here to show on the homepage:

- **about.jpg** — About section (e.g. team photo, cleaning in action). Used in "Who we are".
- **testimonials.jpg** — Testimonials section (e.g. happy customer, clean home, or team). Used in "What our customers say".

You can use `.png` or `.webp`; if you do, update the file names in `src/app/page.tsx` (search for `/home/`).

Recommended size: about **800×600** or similar for a good balance.
