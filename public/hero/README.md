# Hero slider images

Add your **4 hero photos** here with these exact names:

- `slide-1.jpg`
- `slide-2.jpg`
- `slide-3.jpg`
- `slide-4.jpg`

You can use `.png` or `.webp` instead of `.jpg` — if you do, update the file names in `src/data/heroSlides.ts` to match (e.g. `slide-1.png`).

Recommended size: **1920×800** pixels or similar (wide landscape) so they look good on all screens.
